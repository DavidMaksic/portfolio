# Блог Апликација

### Једноставна блогинг апликација са функцијом обележавања чланака. Направљена са ванила JavaScript-ом користећи MVC архитектуру.

<HeroImage
   image={BlogImg}
   imageMobile={BlogMobileImg}
   isResponsive={true}
   title="Блог Апликација"
/>

Мој Блог је ванила JavaScript апликација која омогућава својим корисницима да пишу, мењају и бришу чланке. Направљена је у HTML-у, са ванила CSS-ом и JavaScript-ом. Апликација је структуисана користећи MVC (Model-View-Controller) архитектуру, која раздваја апликацију у три дела зарад лакшег разумевања.
