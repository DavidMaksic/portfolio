# Yakuza Kiwami 2 Вебсајт Редизајн

### Комплетан редизајн промоционалног вебсајта за награђивану видео-игру Yakuza Kiwami 2.

<HeroImage image={YakuzaImg} title="Yakuza Kiwami 2 Вебсајт Редизајн" />

Yakuza Kiwami 2 вебсајт је редизајн заснован на званичном сајту ове видео-игре која је у власништву компаније SEGA. Направљен са HTML-ом, ванила CSS-ом, и са JavaScript-ом за неке функционалности.

<br />

Дизајн сајта је заснован на познатим бојама ове франшизе, са пар CSS трикова који имају за циљ да улепшају сајт. Прилагодљиви дизајн је искоришћен да би омогућио прегледност на многим уређајима.
