# O Casamento

### A brochure website for a fictional wedding company, which showcases what this company offers.

<HeroImage image={CasamentoImg} title="O Casamento" />

O Casamento is a presentational website which showcases what this wedding orgainzation company has to offer. Website is written in HTML along with the CSS extension SASS, with some JavaScript for a few functionalities.

<br />

This website features a very modern look, with eye catching colors. Responsive design is implemented which enables users to browse it on multitude of devices.
