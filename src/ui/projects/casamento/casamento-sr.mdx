# O Casamento

### Вебсајт брошура за фикционалну фирму за венчања, која представља њену понуду.

<HeroImage image={CasamentoImg} title="O Casamento" />

O Casamento је презентациони вебсајт који приказује шта ова фирма за организовање венчања има да понуди. Вебсајт је написан у HTML-у заједно са CSS екстензијом SASS, са мало JavaScript-а за одређене функционалности.

<br />

Овај вебсајт садржи веома модеран изглед са упадљивим бојама. Прилагодљиви дизајн омогућава корисницима да виде сајт на мноштву уређаја.
