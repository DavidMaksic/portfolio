# VS Code Редизајн

### Најпопуларнији IDE модерно редизајниран, без непотребних опција и са лакшом навигацијом.

<HeroImage image={VSCode} title="VS Code Редизајн" />

Овај редизајн је урађен користећи **Custom CSS and JS loader** екстензију направљену од стране be5invis. Све промене су обављене са најобичнијим CSS стиловима. JavaScript није коришћен.

<br />

At the beginning of the article I will share all the extensions I use, along with useful shortcuts which are intuitively made for easy navigation, and settings which hide unnecessary clutter.

На почетку чланка ћу поделити све своје екстензије које користим, пречице на тастатури које су интуитивно изабране зарад лакше навигације, и подешавања која сакривају непотребан садржај.

<ArticleH2 id={'главни-екран'} text={'Главни екран'} />

При активацији нових CSS стилова, корисник ће видети да је са главног екрана уклоњен сав непотребан садржај. Сви помоћни менији (горњи, леви и доњи) су уклоњени остављајући доста више простора за сам код.

<br />

Са леве стране нема ни бројева који обележавају линије кода (ако корисника интересује одређена линија кода, може да је претражи користећи већ постојећи модални прозор, што је много брже).

<ArticleH2 id={'помоћни-менији'} text={'Помоћни менији'} />

Први приказани помоћни мени је мени за **преглед фајлова**. Навигација лево од помоћних менија је сакривена (она се може видети изнад, на главној слици чланка), а за отварање и затварање ових менија (и других екрана, попут подешавања) користе се пречице на тастатури (што чини писање кода неупоредиво бржим, иако испрва захтева мало навике).

<br />

<ArticleImage image={VSCodeImg2} alt="Преглед фајлова" />

<br />
<br />

Модали имају нови изглед, са блажим ивицама и позадинским ефектом замагљеног стакла. Ово важи за све модале (јер су истог типа, само са другачијим садржајем). Приступа им се путем пречица на тастатури.

<br />
