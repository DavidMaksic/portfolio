# VS Code Redesign

### Most popular IDE redesigned for the current year, with no clutter and easier navigation, featuring all of my settings, extensions and shortcuts.

<HeroImage image={VSCode} title="VS Code Redesign" />

This redesign was made possible by using **Custom CSS and JS loader** extension made by be5invis. All changes are made by simply overriding VS Code's CSS styles. No JavaScript was used.

<br />

At the beginning of the article I will share all the extensions I use, along with useful shortcuts which are intuitively made for easy navigation, and settings which hide unnecessary clutter.

<ArticleH2 id={'main-screen'} text={'Main screen'} />

Upon applying redesigned CSS styles, user will see that the main screen with code has basically zero clutter. All sidebars (top, left and bottom) are gone leaving a lot of space for code itself, with no distractions.

<br />

There are no line numbers on the left either (user can search for a specific line using a built-in modal, which is way faster).
