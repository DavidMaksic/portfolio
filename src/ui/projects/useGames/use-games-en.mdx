# useGames App

### App which fetches game data using REST API. Users can browse, rate and save games.

<HeroImage image={UseGamesImg} title="useGames App" />

useGames is a browsing app for finding, rating and saving video-games. It's built with React and Tailwind CSS. RAWG video game API is used for data fetching.

<ArticleH2 id={'searching-for-a-game'} text={'Searching for a game'} />

When user searches for a game, a list of games will be displayed on the left side, from which each game can be selected. On the right, details of the currently selected game are displayed, where user can input data: rating and hours played. User can save the game only if both are specified.
