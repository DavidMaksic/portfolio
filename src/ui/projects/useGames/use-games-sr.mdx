# useGames Апликација

### Апликација која преузима податке о видео-играма користећи REST API. Корисници могу да разгледају, оцене и сачувају игре.

<HeroImage image={UseGamesImg} title="useGames Апликација" />

UseGames је апликација где је могуће тражити, оценити и сачувати видео-игре. Направљена је са React-ом и Tailwind CSS-ом. RAWG API је коришћен за преузимање података о играма.

Када корисник напише име игре, листа игара ће се појавити на левој страни екрана, где свака игра може бити изабрана. Са десне стране су приказани детаљи о тренутно изабраној игри, где се од корисника очекује да оцени игру и да упише колико му је сати било потребно да је пређе. Корисник може да сачува игру само ако је попунио оба податка.
